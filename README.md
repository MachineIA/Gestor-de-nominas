Narrativa Profesional (Contexto / SRS Simplificado)

Servicios Corporativos Caribe SRL, empresa dominicana dedicada a servicios administrativos, 
requiere un sistema computacional sencillo para gestionar su proceso de nómina mensual.

Actualmente, los registros se manejan en hojas de cálculo, lo que genera errores frecuentes, 
duplicidad de datos y cálculos inconsistentes. La gerencia solicita un prototipo funcional
desarrollado en C#, con persistencia en base de datos, capaz de:

• Registrar empleados (Código, Nombre, Departamento, Salario Base).
• Registrar horas trabajadas o datos para calcular el salario mensual.
• Calcular salario neto aplicando deducciones básicas (AFP, ARS, ISR opcional).
• Generar un reporte mensual de la nómina.

Este prototipo servirá como base para una futura versión empresarial.
